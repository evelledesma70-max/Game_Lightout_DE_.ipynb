# Game_Lightout_DE_.ipynb
IA para Lights Out: Juego de lógica donde apagas luces pulsando cuadros que afectan a sus vecinos. El método BFS busca siempre la ruta más corta; el DFS prueba caminos rápidos aunque den vueltas; y el ATS es el más inteligente: usa pistas del tablero para no agotar la RAM. Incluye un análisis de velocidad y memoria de cada algoritmo.
